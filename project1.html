Project Overview

Screenshots of the Application

1. Login Page

The login page provides a simple and secure authentication system, allowing users to safely access their personalized dashboard and data.

2. Home Page

The home page offers an overview of the user's mental health status, showing daily assessments, historical data, and available coping resources.

3. Assessment Page (ChatGPT Integration)

This page allows users to complete daily mental health assessments. The responses are analyzed by the ChatGPT API to determine relapse risk, which then triggers appropriate actions if the risk is medium or high.

4. Coping Techniques Page

The coping techniques page provides users with personalized strategies to manage cravings and stress. It includes suggestions for mindfulness exercises, distraction methods, and emergency contacts for additional support.
