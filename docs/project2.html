Project Overview

This ADHD Assistant was inspired by my personal experience managing ADHD, where I aimed to create an application that helps people facing similar challenges with task management, focus, and organization. The app provides practical tools that support day-to-day productivity, including task management, a focus timer, and a note-taking system.

Features and Benefits

1. Task Management System

Functionality: Users can create, update, and delete tasks, assign priority levels (low, medium, high), and set due dates.

Technologies: C++ for backend logic, SQLite for data storage, HTML/CSS for the user interface.

Why It's Good: Helps users organize tasks effectively with emphasis on priority and due dates, reducing cognitive load by providing a structured system.

Skills Demonstrated: Database management (CRUD operations in SQLite), backend development with C++, frontend design for intuitive task management.

2. Focus Timer

Functionality: Implements a Pomodoro-style timer, breaking work into focused intervals with short breaks.

Technologies: C++ for backend calculations, HTML/CSS/JavaScript for the timer interface.

Why It's Good: Encourages productivity, especially for users with ADHD, through structured work and break intervals, helping them stay focused.

Skills Demonstrated: C++ backend for managing timer intervals, JavaScript for dynamic frontend interactions, UX/UI design for simple, effective visuals.

3. Note-Taking System

Functionality: Allows users to take and store notes quickly, offering a space for capturing thoughts, ideas, or reminders.

Technologies: C++ for backend logic, SQLite for persistent note storage, HTML/CSS for the interface.

Why It's Good: Helps manage scattered thoughts—common for individuals with ADHD—by providing an accessible space for note-taking.

Skills Demonstrated: Integration of note-taking with SQLite for data persistence, backend handling of user inputs in C++, frontend interface design.
