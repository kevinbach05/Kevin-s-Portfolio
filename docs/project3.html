Project Overview

This project was developed for a local photo booth business, providing them with a user-friendly and interactive website. The website includes a carousel for showcasing video work, a booking system with multiple package options, and a contact page integrated with the SendGrid API for seamless communication. The site was built to improve the business's online presence, making it easy for users to book services, view their work, and reach out for inquiries.

Features and Technologies

  • Carousel: A video carousel that displays work from the photo booth, allowing users to see past events and client work. (Screen recording)
  • Booking System: A fully functional booking system that allows users to select from multiple service packages and book appointments directly through the site.
  • Contact Page: Integrated with SendGrid API to ensure users can easily get in touch with the business for inquiries and bookings.
  • Responsive Design: The entire website is optimized for both mobile and desktop viewing to ensure users can access the services from any device.
  • Technologies Used: HTML, CSS, JavaScript for frontend, SendGrid API for contact form, and a dynamic carousel for video presentation.

Screenshots and Media of the Application

1. Home Page

The home page features a clean and modern design, providing users with quick access to information about the photo booth services and links to the booking system.

2. Carousel of Videos

The carousel showcases video clips of past events and client work, giving potential customers a visual representation of the business's offerings.

3. Book Now & Packages

The booking page allows users to browse through different service packages and book their desired option directly on the site, making the booking process seamless and user-friendly.

4. Contact Page

The contact page is integrated with the SendGrid API, allowing users to send inquiries and booking requests directly to the business through a simple form.
